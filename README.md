# Desenvolvendo o Dio Bank Backend (API)

Desenvolvido por [Alef A. Araujo](https://github.com/3alyef)

#### Projeto desenvolvido com NodeJs e Express

### Tecnologias
- Typescript
- NodeJs
- Express

### Como rodar o projeto

1 - Clone o repositório

2 - Instale as dependeências
    
    npm install

3 - Crie a build

    npm run build

3 - Execute o projeto

    npm start
